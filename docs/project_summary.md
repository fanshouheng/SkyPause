# 项目更新说明

## 最新功能
- 实现实时变速录制（0.05x 到 2.0x）
- 添加悬浮调速控制
- 高质量视频录制（60fps，720p）
- 前台服务通知

## 技术特点
- 使用 MediaRecorder 实现高质量录制
- 实时帧率调整实现变速效果
- 悬浮窗实现速度控制
- 完全合法且安全的实现方式

## 使用说明
1. 点击悬浮按钮开始录制
2. 使用悬浮调速控制调整速度
3. 再次点击按钮停止录制
4. 查看通知了解录制状态
5. 在相册中查看录制的视频

## 注意事项
- 录制时请保持屏幕亮起
- 速度调整范围：0.05x - 2.0x
- 视频保存在应用专属目录中
